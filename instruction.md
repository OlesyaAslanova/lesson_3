# Инструкция по языку Markdown

## Стилизация текст

**Полужирный** или __полужирный__
_курсив_ или *курсив*

## Цитирование текста
> Первый уровень цитирования
>> Второй уровень цитирования
>>> Третий уровень цитирования 

## Списки
### Нумерованные списки
1. Пункт первый
2. Пункт второй
3. Пункт третий 
### Ненумерованные списки
* Первый элемент
* Второй элемент
+ Третий элемент
